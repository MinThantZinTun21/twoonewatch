Watch Along — အသုံးပြုလမ်းညွှန်

ဒီစာမျက်နှာမှာ Watch Along ကို အသုံးပြုနည်းနဲ့ ပြဿနာဖြေရှင်းနည်းများကို ရှင်းပြထားပါတယ်။

အခြေခံအကြောင်းအရာ

  • Man Utd ပွဲစမယ့်အချိန် — Watch Along icon ကို နှိပ်လိုက်ရင် Min Min Htun Live သွားတက်ပါမယ်။
  • Multiple Match — တပြိုင်နက် ၄ ပွဲထိ ကြည့်နိုင်ပါတယ်။
  • ဖုန်း Screen — ဗီဒီယို fix မဖြစ်သေးရင် ထောင့်နားက icon ကို နှိပ်ပြီး view ပြောင်းနိုင်ပါတယ်။
  • Watch Along Icon — အနီရောင်နဲ့ လူ ၂ ယောက် ပါတဲ့ ခလုတ်ဖြစ်ပါတယ်။
  • အသစ်ထည့်စွက် — Chelsea ပွဲလည်း အလားအလာရှိပါက [date] တွင် Watch Along ထည့်ပေးပါမယ်။

⚠️ သေချာစွာ VPN ပိတ်ထားမှ Watch Along အလုပ်လုပ်ပါမယ်။ VPN သုံးထားလျှင် Live မပေါ်နိုင်ပါ။

Development အခြေအနေ

လက်ရှိ Development လုပ်ဆောင်ဆဲ ဖြစ်ပြီး error တွေ အများကြီးဖြစ်နိုင်ပါတယ်။

Man Utd ပွဲလာတဲ့အချိန် Min Min Htun Live မပေါ်လာလို့ ဘယ်ဟာနှိပ်ရမလဲ မသိသေးပါက — ရှိတဲ့ ခလုတ်တွေကို တစ်ခုပီးတစ်ခုလိုက်နှိပ် ကြည့်ပါ။

ပြဿနာဖြေရှင်းခြင်း (Troubleshooting)

  • Error တက်တယ် — VPN ကို ပိတ် (disable) လုပ်ကြည့်ပါ။
  • Live မပေါ်တယ် — Watch Along icon ကို ပြန်နှိပ်ပါ၊ မရသေးရင် အခြား ခလုတ်များကို တစ်ခုပီးတစ်ခု စမ်းနှိပ်ပါ။
  • Network Slow — Wi-Fi ပြောင်းခြင်း/Internet ပြန်ဖွင့်ခြင်းနဲ့ ကြိုးစားပါ။
  • Update App — App version နောက်ဆုံးမဟုတ်လျှင် Store မှာ update လုပ်ပါ။
